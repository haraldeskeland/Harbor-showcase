# Harbor Screenshots

This directory contains screenshots and images for the Harbor showcase.

## Directory Structure

```
images/
├── screenshots/
│   ├── landing-page.png
│   ├── dashboard.png
│   ├── orders.png
│   ├── inventory.png
│   ├── kanban-board.png
│   └── margins.png
└── README.md
```

## Adding Screenshots

To add screenshots to the showcase:

1. Place your images in the `screenshots/` subdirectory
2. Use descriptive filenames (e.g., `landing-page.png`, `dashboard.png`)
3. Update the main README.md to reference these images
4. Recommended image formats: PNG or JPG
5. Recommended dimensions: 1920x1080 or similar high-resolution formats

## Image Guidelines

- Use high-quality screenshots
- Ensure sensitive data is removed or masked
- Keep file sizes reasonable (compress if necessary)
- Use consistent styling across screenshots
